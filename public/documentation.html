Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of all movies to the user. /movies GET NONE NONE A JSON object containing all movie data.
Return data (description, genre, director, image URL) about a single movie by title to user. /movies/:Title GET :Title NONE A JSON object holding data about the selected movie containing description, genre, director and img URL. Example: 
							
								{
									Title: '',
									Description: '',
									Genre: {
										Name: '',
										Description: ''
									},
									Director: {
										Name: '',
										Birth: '',
										Death: '',
										Bio: ''
									},
									Actors: [],
									ImagePath: ''
								}
Return data about a genre (description) by name/title.(e.g. "Thriller") /movies/:Genre GET :Genre NONE A JSON object holding data about the selected genre. Example: 
							
								{
									Name: '',
									Description: ''
								}
Return data about a director (bio,birth year, death year) by name. /movies/directors/:Name GET :Name NONE A JSON object holding data about the selected director. Example: 
							
								{
									Name: '',
									Birth: '',
									Death: '',
									Bio: ''
								}
Allow new users to register. /users POST NONE A JSON object holding data about the user that's about to be added. Example:' 
							
								{
									Username: '',
									Email: '',
									Password: '',
									Birthday: '',
									Favorites: [],
									ID: ''
								}
A JSON object holding data about the user that has been added with unique ID. Example:' 
							
							{
								Username: '',
								Email: '',
								Password: '',
								Birthday: '',
								Favorites: [],
								ID: ''
							}
Allow registered users to login to application. /login POST :Username, :Password NONE A JSON object containing information about the user and a JWT token.
Allow useres to update their user info (username). /users/:Username PUT :Username A JSON object containing new username. Example: 
							
								{
									Username: '',
									Email: '',
									Password: '',
									Birthday: '',
									Favorites: [],
									ID: ''
								}
A JSON object containing the updated user profile. Example: 
							
								{
									Username: '',
									Email: '',
									Password: '',
									Birthday: '',
									FavoritesMovies: [],
									ID: ''
								}
Allow users to add a movie to their list of favorites. /users/:Username/movies/:MovieID POST :Username, :MovieID NONE A message indicating the movie was added to favorites list.
Allow users to delete a movie from their list of favorites. /users/:Username/movies/:MovieID DELETE :Username, :MovieID NONE A message indicating the movie was removed from favorites list.
Allow exsisting users to deregister. /users/:Username DELETE :Username NONE A message indicating the user was deregistered.
